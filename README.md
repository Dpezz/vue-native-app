# vue-native-app
This is a starter template for vue native projects.

## How to use this template

Required:
* node >= 6.0
* npm >= 4.0
* React Native CLI
* Vue Native CLI

```bash
$ git clone <projectName>
$ cd <projectName>
$ npm install
$ npm start || npm run ios || npm run android
```
